# Library Management System

## Project Description
This project is a Library Management System designed to manage books, users, and transactions efficiently.
It helps librarians to issue books, return books, and track availability.

## Features
- Add new books
- Update book details
- Issue books to users
- Return books
- Track available and issued books

## Technologies Used
- Python
- MySQL
- SQL Queries
- GitHub

## Database Tables
- Books
- Users
- Issued_Books

## Operations Performed
- INSERT
- UPDATE
- DELETE
- SELECT
- JOIN queries

## Future Enhancements
- Web-based interface
- User login system
- Fine calculation for late returns
